There is an installtion file where database table will be auto created and droped based on activation and deactivation of the plugin.

Main shortcode is [folders] user can create new page and paste it on the page from admin area, which is used to show layout and form where can add new folders and subfolders.

[folders] main shortcode
[folderoptions] is used to show all folders name dropdown where can make subfolders of any folder.
[folderlist] is used to show all folders grid on the front end where can check folders and subfolder working mode
[subfolderlist] is used to load subfolders of parent folders. 

all functionality ajax based.
